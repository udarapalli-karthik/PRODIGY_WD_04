Machine Learning

AI Image Generator

Problem Statement

Creating high-quality artistic images from text descriptions manually requires skilled designers and significant time. Small businesses and content creators lack access to affordable, fast, and customizable image generation tools. This project aims to bridge that gap using AI-powered generative models that produce unique images from simple text prompts.

How It Was Built

The core model is a Generative Adversarial Network (GAN) trained on a diverse dataset of artistic images. A text encoder (based on CLIP) converts the user's prompt into a latent vector, which the generator network uses to synthesize a new image. The discriminator was trained adversarially to ensure photo-realistic quality. A FastAPI backend serves the model, and a React.js frontend renders the results in real time.

Tools & Technologies

Python PyTorch CLIP (OpenAI) GAN Architecture FastAPI React.js CUDA / GPU Hugging Face

Applications & Uses

  • Generating marketing and advertising visuals instantly
  • Design prototyping for UI/UX teams
  • Digital art creation for artists and content creators
  • Game asset and concept art generation
  • Educational tools for visual learning materials
Web Development

Weather Dashboard

Problem Statement

Users often need to switch between multiple weather services to get real-time conditions, historical trends, and multi-city forecasts. There was no single platform offering an interactive, visually rich, and fast weather experience. This dashboard solves that by aggregating live weather data into one clean, responsive interface.

How It Was Built

The frontend was built using React.js with dynamic components for current conditions, 7-day forecasts, and hourly breakdowns. The OpenWeatherMap API provides live weather data. Chart.js renders temperature and precipitation graphs, while Leaflet.js powers the interactive weather map. A lightweight Node.js/Express backend manages API keys securely and caches responses to optimize performance.

Tools & Technologies

React.js Node.js Express.js OpenWeatherMap API Chart.js Leaflet.js CSS3 / Glassmorphism REST APIs

Applications & Uses

  • Real-time personal weather monitoring for any city
  • Travel and trip planning with multi-day forecasts
  • Agricultural use — planning irrigation and harvest timings
  • Outdoor event management and scheduling
  • Research tool for historical climate data analysis
Machine Learning

Customer Sentiment Analyzer

Problem Statement

Businesses receive thousands of customer reviews daily across e-commerce platforms, social media, and support tickets. Manually reading and categorizing each review as positive, negative, or neutral is completely impractical at scale. This project automates that process using NLP — enabling businesses to instantly understand customer sentiment and take data-driven decisions.

How It Was Built

A BERT (Bidirectional Encoder Representations from Transformers) model was fine-tuned on a labeled product review dataset using Hugging Face Transformers. The model classifies input text into Positive, Negative, or Neutral categories with confidence scores. A FastAPI REST API exposes the inference endpoint, and a React.js dashboard visualizes results with donut charts, keyword clouds, and a real-time review feed.

Tools & Technologies

Python BERT / Transformers Hugging Face FastAPI React.js Pandas scikit-learn Chart.js

Applications & Uses

  • E-commerce platforms — auto-tagging product reviews
  • Brand reputation monitoring across social media
  • Customer support ticket prioritization by urgency
  • Product improvement insights from negative feedback
  • Market research and competitor analysis
